# Stopwatch Web App

A sleek and stylish web application for a stopwatch with start, stop, reset, and lap functionalities.

## Usage

1. Open [index.html](./index.html) in a web browser.
2. Use buttons to control the stopwatch:
   - Start
   - Stop
   - Reset
   - Lap
